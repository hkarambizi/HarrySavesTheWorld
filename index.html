Portfolio

fifo App


I created fifo with Ruby on Rails as an enterprise app for an Atlanta festival concessions company to manage and track inventory flow, eliminating overstocking and cutting over $1,000 per event day on printing and added labor costs. Although this app was my creation, I worked in a group of three using Agile programming methodologies to develop this.


Cee-Lo Dice Game


A fun and interactive online dice game to play with friends. The game allows you roll three dice and achieve a score for your roll. The next player then rolls until you find a winner. Languages used: HTML5, CSS, and JavaScript/JQuery


WorkBox - Job Hunt Tracker


WorkBox is a full CRUD MEN Stack app designed as a job search toolbox. I created this Beta version as a networking contact relationship manager. Languages used: HTML, JavaScript, JQuery, NodeJs, MongoDB.


Plate Swap App


Using an Agile workflow and pair-programming, a group of three developers, in which I was a part of, built a MEAN stack application for the Meal Swap community to easily post and claim swaps online. Languages used: Angular JS, Bootstrap, Express, MongoDB, Node.JS.


City Tips App


City Tips is a web app created as a group project to master Agile programming methodologies to manage workflow to allow for any neccessary pivots or added features.
